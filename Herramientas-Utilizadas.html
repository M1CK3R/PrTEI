Pero, como hicimos todo esto?

En nuestro equipo ninguno teniamos experiencia en desarrollo web, finanzas o desarrollo de chat-bots​.

Teniamos mucho trabajo y muy poco tiempo, pero gracias a las herramientas que nos proporciona internet, logramos cumplir con nuestro proyecto, a continuacion se describira cada una de las herramientas que utilizamos.

Crear Pagina Web

La pagina web se hizo desde 0, sin utilizar ninguna plantilla, ya que no encontramos alguna que cumpliera con todos nuestros requisitos y que mantuviera el estilo que nesesitabamos, y tambien por las limitaciones que nos dan ese tipo de desarrollo web.

Editor de Codigo

Para empezar a desarrollar la pagina web primero se nesesita de un Editor de Codigo​, en este caso utilizamos Sublime Text 3, ya que es sencillo y facil de usar, ademas del bajo consumo de recursos.

Codigo HTML

Principalmente el codigo se creo investigando en diversos sitios sobre programacion y desarrollo web, y con ayuda de ChatGPT para que nos solucionara dudas que pudieran surgir o para querer implementar funciones.

Recursos Externos

Principalmente iconos, logos, imagenes y otros recursos que se agregaron a la pagina que no fueron hechos por nosotros.
Iconos Vectoriales: flaticon.es
Imagenes y Logos: vecteezy.es

Crear CHAT-BOT

El CHAT-BOT se desarrollo con Blackbox.ia una pagina que te ayuda a desarrollar un asistente de ia propio, se entrena con paginas web que seran sus bases de conocimiento para generar las respuestas a tus preguntas.

Proceso de Prueba

Aca se prueba el CHAT-BOT para ver si funciona correctamente, a pesar de que esta plataforma es en ingles, nos da respuestas acertadas, se implemento a la pagina web por medio de un <iframe>.
